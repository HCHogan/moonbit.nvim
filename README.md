# moonbit.nvim

Neovim support for the [Moonbit language](https://www.moonbitlang.com).

## Roadmap

- [ ] Tree-sitter support:
  - [x] Highlights
  - [ ] Folds
  - [ ] Indents
- Build system support:
  - [ ] Compiler plugin

## Installation

### `lazy.nvim`

```lua
{
  'tonyfettes/moonbit.nvim',
  ft = { 'moonbit' }
  opts = {},
}
```
